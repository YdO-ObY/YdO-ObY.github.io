## 2.4 무치형(Void)
    아직은 자세히 설명이 안되는 부분이 있는거같다.
    더 깊이 들어가려면 포인터를 배워야하는것으로 보인다.
    일단 이런게 있구나 생각만 해놓자!


## 예시
```c++
#include <iostream>

int main() {
    
    
    //void num;
    //void 변수는 선언이 불가능하다.
    
    int i = 123;
    float f = 123.456f;

    //* 포인터
    void *my_void;
    //주소를 나타내는 코드, 지금은 몰라도 된다.
    //int형 주소 타입으로 변경
    my_void = (void*)&i;
    
    //float형 주소 타입으로 변경
    my_void = (void*)&f;
    
    //포인터 잠깐! 예습
    //데이터 타입, 사이즈가 다르더라도 주소를 나타내는 데이터양은 같다.??
}
```
